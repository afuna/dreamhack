<DW::Logic::Importer-get_import_data( $u, id1, [ id2, id3 ] )>>

Get import data for this user for all provided import ids

<DW::Logic::Importer-get_import_data_for_user>>

<DW::Logic::Importer-get_import_items( $u, id1, [ id2, id3... ] )>>

Get import items for this user for all provided import ids.

<DW::Logic::Importer-get_import_items_for_user( $u, id1, [ id2, id3... ] )>>

<DW::Logic::Importer-get_queued_imports( $u )>>

Get a list of imports that have yet to be processed. May be in the schwartz queue, and thus running soon or else in the import queue, and have yet to be put into the schwartz queue. The latter may not run if they are duplicates of something in the schwartz queue.
