NAME

DW::Worker::ContentImporter::Local::Userpics - Local data utilities for userpics

Userpics

These functions are all part of the Saving API and handle saving data to the local site. These are not to be called outside of the import pipeline.

$class->import_userpics( $user, $errors, $default, $userpics )

Import the following userpics, default first, but only import the other ones if they *all* fit. This will return an array of userpic IDs that were imported.

$errors is an arrayref that errors will be appended to.

$class->import_userpic( $user, $errors, $userpic )

$userpic is a hashref representation of a single icon, with the following format:

  {
    url => 'http://some.tld/some.jpg', # URL to image
    default => 0,                      # Is this the default image?
    keywords => [
        'keyword',
        'another keyword',
    ],
    comment => 'This is my icon!',     # The comment for the icon
  }

$errors is an arrayref that errors will be appended to.

This will return 0 if it failed, 1 if it suceeded, and 2 if it was an existing pic.
