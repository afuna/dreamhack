NAME

DW::Routing - Module to allow calling non-BML controller/views.

Page Call API

$class->call( $r, %opts )

Valid options:

uri - explicitly override the uri =item role - explicitly define the role =item username - define the username, implies username role =item ssl - this is a SSL page, implies the ssl role

This method should be directly returned by the caller if defined.

$class->get_call_opts( $r, %opts )

Valid options:

uri - explicitly override the uri =item role - explicitly define the role =item username - define the username, implies username role =item ssl - this is a SSL page, implies the ssl role

Returns a call_opts hash, or undefined.

$class->call_hash( $class, $call_opts )

Calls the raw hash.

Registration API

$class->register_static( $string, $filename, %opts )

Static page helper.

string - path
filename - template filename
Opts ( see register_string )

$class->register_string( $string, $sub, %opts )

string - path
sub - sub
Opts:
args - passed verbatim to sub.
ssl - If this sub should run for ssl.
app - 1 if app
user - 1 if user
format - What format should be used, defaults to HTML
formats - An array of possible formats, or 1 to allow everything.

$class->register_redirect( $string, $dest, %opts )

Redirect helper.

string - path
dest - destination
Opts ( see register_string )
keep_args - Persist GET arguments over redirect ( same as the keep_args argument to create_url ).
full_uri - A full URI ( http://...../foo v.s. /foo )

$class->register_regex( $regex, $sub, %opts )

regex
sub - sub
Opts ( see register_string )
