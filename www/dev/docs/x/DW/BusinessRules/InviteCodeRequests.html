NAME

DW::BusinessRules::InviteCodeRequests - business rules for invite code requests handling

SYNOPSIS

  my $can_request = DW::BusinessRules::InviteCodeRequests::can_request( user => $u );

API

DW::BusinessRules::InviteCodeRequests::can_request( user => $u )

Return whether the user can make a request for more invite codes. Default implementation allows the user to make a new request if they have no unused invite codes, they have no pending requests for review, and are not sysbanned from using the invites system.
