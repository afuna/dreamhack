NAME

DW::Panel - Generic movable container which wraps around an object of class LJ::Widget, and remembers state and position.

SYNOPSIS

  use DW::Panel;

  my $panels = DW::Panel->init( u => $remote );  
  $panels->render_primary;
  $panels->render_secondary;

API

$class->init( [ u => $u] ) Class method; initializes the panels with their settings, etc, for this user.

$object->render_primary Render the widgets that belong in the primary column

$object->render_secondary Render the widgets that belong in the secondary column

$object->_render( widgetname ) Render the widget and its container.
